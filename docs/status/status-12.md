# status-12: 自然対流ソルバー安定性改善 — adaptive dt / RC energy / 収束判定修正

[<- status-index](status-index.md) | [<- README](../../README.md)

## 日付

2026-04-01

## 概要

status-11 のフォーカスガード（チェッカーボード温度・高温位置ずれ・流速逆転）に対し、
段階的デバッグにより原因を特定し、3点の修正を実施。

## 診断結果

| テスト | 結果 | 判定 |
|--------|------|------|
| 差分加熱キャビティ Ra=1000（q_vol不使用） | T正常、v方向正しい | OK |
| q_vol 純粋伝導（beta=0, g=0） | T_max中心、対称、v=0 | OK |
| q_vol+浮力（人工物性 mu=0.01） | T_max中心、v>0（上向き） | OK |
| q_vol+浮力（mu=1e-4） | T_max位置ずれ、v<0（逆転）| NG |
| q_vol+浮力（空気実物性 mu=1.85e-5） | T非物理的、mass残差大 | NG |

### 破綻点の特定

粘性を段階的に下げたスイープにより、**mu=1e-4** で破綻することを特定。

- mu=1e-3: Pe=0.01, CFL=0.03 → **正常**
- mu=1e-4: Pe=25, **CFL=10** → **異常**（T_max位置ずれ、v逆転）

根本原因: **CFL数 >> 1** により各タイムステップでのSIMPLE反復が収束せず、
累積誤差が物理場を破壊。

### 追加発見: SIMPLE収束判定の問題

非定常解析でエネルギー方程式のRHS = ρCp/dt × T_old（≈ 1.19e5 × 300 = 3.6e7/cell）が
支配的で、q_vol（500 W/m³）の寄与は0.4%。このため相対残差が常に~1.0となり、
温度残差がSIMPLE収束判定を支配して「未収束」と判定される誤動作が発生。

### 修正詳細

1. **CFL制約付き適応的時間刻み** (`_adaptive_dt`)
   - 各ステップ開始前に CFL_max=0.5 に基づくdtを計算
   - dt = min(dt_user, CFL_max × dx / |v_max|) を各方向で評価
   - frozen dataclass のため dt 差し替え時に新 Input を生成

2. **エネルギー方程式のRhie-Chow面速度**
   - `build_energy_system()` に `rc_face_velocities` パラメータ追加
   - RC面速度指定時は面ベースの対流フラックスを計算（チェッカーボード抑制）
   - 未指定時は従来通りセル中心速度の線形補間（後方互換）

3. **SIMPLE収束判定の温度除外** (`_simple_convergence_residual`)
   - 温度(T)を収束判定から除外し、u, v, w, p, mass のみで判定
   - 定常・非定常の両方で適用

## 残存問題

空気実物性（mu=1.85e-5）のq_vol+浮力問題は修正後も以下が残る:
- mass残差が O(1-100) と大きく、各ステップ内で質量保存が不十分
- 長時間実行で流速方向が不安定化（v < 0 が発生）
- 温度場のチェッカーボードは改善したが完全には解消されていない

### 根本原因の考察

空気のPr≈0.71、Re_grid = v×dx/ν ≈ 0.015×0.0025/1.57e-5 ≈ 2.4 で、
セルレイノルズ数が2を超えている。1次風上差分の安定域だが、
**SIMPLE法の圧力-速度連成がこのレジームで十分に収束していない**。

改善の方向性:
- SIMPLEC（圧力補正の緩和係数を大きくできる）
- PISO（圧力補正の2段階補正）
- 非直交補正の圧力補正方程式への適用
- より保守的な緩和係数（alpha_u=0.1, alpha_p=0.03）
