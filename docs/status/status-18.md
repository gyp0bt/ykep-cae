# status-18: 1D過渡ジュール電熱 Gauss-Seidel 検算スクリプト

## 日付

2026-04-07

## 概要

区間ごとに物性値と断面積が異なる1次元固体ジュール加熱を、
制御体積法 + 陰的Euler + Gauss-Seidel反復で解く検算スクリプトを作成。

手元のスクリプトで均一物性でもメッシュ依存性が出るバグの検証が目的。
ローテク志向でワンスクリプトにまとめた。

## 実施内容

### 1. 検算スクリプト `scripts/verify_1d_joule_gauss_seidel.py`

**物理モデル:**
```
ρcA ∂T/∂t = ∂/∂x [kA ∂T/∂x] + I²ρ_e(T)/A
ρ_e(T) = ρ_e0(1 + α(T - T_ref))
```

**離散化のポイント（メッシュ依存性回避）:**
- 面コンダクタンスに**直列抵抗モデル**を使用:
  `1/C_{i+1/2} = dx_i/(2·k_i·A_i) + dx_{i+1}/(2·k_{i+1}·A_{i+1})`
- ジュール発熱の体積積分: `q_vol × A × dx` （Aを忘れない）
- 過渡項: `ρ·c·A·dx/dt` （Aを忘れない）
- 境界面: 半セル距離 `dx/2` を使用

### 2. テスト結果

| テスト | 結果 | 詳細 |
|--------|------|------|
| Test 1: 均一ロッド定常解 | PASS | 2次収束（次数=2.01）、解析解との誤差 O(dx²) |
| Test 2: 銅-ニクロム-銅 | PASS | エネルギー収支相対誤差 5.7e-7 |
| Test 3: 温度依存抵抗率 | PASS | 2次収束確認、収支誤差 1.8e-5 W |
| Test 4: 溶断到達 | PASS | A=1e-8 m², I=5A で 8.49ms に 1000K 到達 |

### 3. メッシュ依存性バグの典型原因

均一物性・均一断面積でも発生しうる原因：
1. **過渡項の A 抜け**: `ρ·c·dx/dt` (誤) vs `ρ·c·A·dx/dt` (正)
2. **発熱項の A 抜け**: `q_vol·dx` (誤) vs `q_vol·A·dx` (正)
3. **境界面の距離**: `dx` (誤) vs `dx/2` (正)
4. **面コンダクタンス**: `k/dx` (誤) vs `kA/dx` (正)

1次元で A が定数の場合、分母と分子で A が打ち消すように見えるが、
Dirichlet 境界条件の処理で `C_bnd = kA/(dx/2)` を使うため、
**内部セルと境界セルで A の扱いが不整合になる**とメッシュ依存性が出る。
