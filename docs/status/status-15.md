## 日付

2026-04-02

## 概要

新ソルバー機能(PISO/SIMPLEC/TVD/BDF2/対流流出BC)を用いた空気実物性
(mu=1.85e-5)の収束評価を実施。PISOの速度緩和バグを修正。

## PISO 速度緩和バグの修正

### 問題

PISO の `_simple_iteration` 内で速度の under-relaxation をスキップしていた。
PISO 本来は outer=1 回で使うため不要だが、`max_simple_iter > 1` の場合は
速度緩和なしだと発散する。

### 修正

全連成手法(SIMPLE/SIMPLEC/PISO)で velocity relaxation を適用するよう統一。
PISO で outer=1 を使いたい場合は alpha_u=1.0 を指定すればよい。

## 収束評価結果

### 条件

- 2cm 差分加熱キャビティ (L=0.02m, T_hot=310K, T_cold=300K)
- 10×10×3 格子
- 空気実物性: rho=1.19, Cp=1007, k=0.026, beta=3.3e-3

### 粘性スイープ (SIMPLE, dt=1e-3, 10ステップ)

| mu | mass残差 | Re_cell 概算 | 判定 |
|----|---------|-------------|------|
| 1e-2 | 1.2e-3 | < 0.01 | **良好** |
| 1e-3 | 1.2e-2 | ~0.1 | 許容範囲 |
| 1e-4 | 5.2e-2 | ~2 | 不良 |
| 1.85e-5 | 6.8e-2 | ~10 | 不良 |

**mass残差はmuに反比例して増大。** セルレイノルズ数が1を超えると急激に悪化。

### 連成手法比較 (mu=1.85e-5, dt=1e-3, 10ステップ)

| 手法 | mass残差 |
|------|---------|
| SIMPLE | 6.78e-2 |
| SIMPLEC | 6.78e-2 |
| PISO | 6.78e-2 |

**3手法でほぼ同一の mass残差。** 圧力-速度連成手法の差は問題の本質ではない。

### メッシュ細分化 (20×20×3, mu=1.85e-5)

| 格子 | mass残差 |
|------|---------|
| 10×10 | 6.78e-2 |
| 20×20 | 1.99e-2 |

メッシュ細分化で **3.4倍改善**。セルレイノルズ数低減が効いている。

### 超保守的条件 (alpha_u=0.05, alpha_p=0.01, dt=1e-4)

- **mass残差: 3.8e-6 → 収束！**
- ただし 5ステップ(0.0005秒)しか進んでおらず、温度場は未発達

## 診断

### 根本原因

空気実物性での問題は **圧力-速度連成手法ではなく、セルレイノルズ数** が原因。

```
Re_cell = v * dx / ν
dx = L/nx = 0.02/10 = 0.002m
ν = mu/rho = 1.85e-5/1.19 = 1.55e-5 m²/s
v ≈ 0.005 m/s (浮力駆動)
Re_cell ≈ 0.005 * 0.002 / 1.55e-5 ≈ 0.6
```

Re_cell < 2 だが、**mass残差の主因は圧力補正方程式の不十分な収束**。
alpha_p=0.1 では圧力補正の1ステップあたりの修正量が小さすぎる。
alpha_u=0.05, alpha_p=0.01 では逆に収束するが非常に遅い。

### 改善方針

1. **メッシュ細分化**: 20×20 で 3.4倍改善。30×30 以上で mass残差 < 1e-2 を期待
2. **outer反復回数の増加**: max_simple_iter=200+ で確実に収束する可能性
3. **適応的緩和**: 残差が小さくなったら alpha_p を増加させる戦略
4. **AMG前処理の圧力方程式への適用**: 現在のILU前処理BiCGSTABから改善
